# Todoist Clone - Modern To-Do List Application

A high-performance, minimalist to-do list application inspired by Todoist, built with React, TypeScript, and TailwindCSS.

## ✨ Features

- **Project Management**: Create, edit, and organize tasks into custom color-coded projects.
- **Task Management**: Full CRUD operations with inline editing, priority levels (P1-P4), and completion tracking.
- **Date & Time Support**: Set due dates and specific times with a smart local-timezone picker.
- **Time Left Countdown**: Real-time relative deadlines (e.g., "2h left", "3d left") with overdue highlighting.
- **Navigation & Filtering**: 
  - **Inbox**: View all active tasks.
  - **Today**: Focus on tasks due today.
  - **Upcoming**: Plan ahead with future tasks.
  - **Completed**: Review your productivity history.
- **Keyboard Shortcuts**:
  - `Q`: Toggle "Add Task" form.
  - `G` then `I`: Go to Inbox.
  - `G` then `T`: Go to Today.
  - `G` then `U`: Go to Upcoming.
  - `Esc`: Cancel/Close modals.
- **Dark Mode**: Seamlessly toggle between light and dark themes with system preference detection.
- **Data Portability**: Export your entire workspace (tasks and projects) as a JSON file.
- **Persistent Storage**: All data is saved locally in your browser.

## 🛠️ Tech Stack

- **Frontend**: [React 18](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Hotkeys**: [react-hotkeys-hook](https://react-hotkeys-hook.vercel.app/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/zhentao-han/Todolist_demo.git
   cd Todolist_demo
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in your browser**:
   Navigate to `http://localhost:5174` (or the port specified in your terminal).
