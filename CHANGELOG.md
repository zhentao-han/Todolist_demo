# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-12
### Changed
- Removed "Inbox" as a default category to simplify project organization.
- Updated main view to dynamically show "All Tasks", "Today", "Upcoming", or "Completed" titles.
- Refined navigation logic in Sidebar and App components.

## [1.1.0] - 2026-03-12
### Added
- Project editing and deletion functionality in the Sidebar.
- Enhanced Task Name editing with explicit Save/Cancel buttons.
- Keyboard shortcuts: `Q` for new task, `G+I/T/U` for navigation.
- Export functionality to JSON for data portability.
- Live countdown "time left" indicator for tasks.
- Priority selection (P1-P4) during task creation.
- Time support in due dates with local timezone fixes.

## [1.0.0] - 2026-03-12
### Added
- Initial release of Todoist Clone.
- Core task management (Add, Toggle, Delete).
- Project/Category organization.
- Basic Due Date support.
- Dark/Light mode support.
- Persistent local storage.
- Responsive design with TailwindCSS.
